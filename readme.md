# Smart Sport Shop

An intelligent sports activity recommendation and e-commerce system that helps users discover the most suitable sports based on their lifestyle, preferences, goals, and physical condition.

The application includes a personalized evaluation questionnaire that analyzes multiple criteria such as daily schedule, fitness level, activity preferences, budget, training availability, and personal objectives. Based on the collected data, the system calculates compatibility scores and recommends one or more suitable sports activities.

After identifying the recommended sports and the user’s experience level (beginner, intermediate, or advanced), the platform automatically suggests appropriate sports equipment and product packages while considering budget constraints and stock availability.

Main features:

- User registration and profile management
- Lifestyle and fitness evaluation questionnaire
- Intelligent sport recommendation system
- Compatibility score calculation
- Beginner/intermediate/advanced level detection
- Sports equipment and product recommendations
- Product, category, and stock management
- Order placement and purchase history
- Recommendation history tracking
- Adaptive recommendations based on user evolution and updated preferences

The project combines database logic, recommendation algorithms, and e-commerce functionalities into a personalized sports assistant platform.
